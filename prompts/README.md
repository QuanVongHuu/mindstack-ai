# Prompts

These prompts power the mindstack-ai workflow and are public templates adapted from a real Claude Cowork daily assistant workflow.

Start here:

1. Read [`personalization-checklist.md`](personalization-checklist.md).
2. Optional: use [`one-prompt-installer.md`](one-prompt-installer.md) to generate a complete setup for Claude Cowork or OpenCode.
3. Copy [`morning-routine.md`](morning-routine.md) into Claude Cowork.
4. Replace the user-input placeholders listed in the checklist.
5. Run once manually.
6. Copy and schedule [`evening-reflection.md`](evening-reflection.md).

For a filled fake morning prompt, see [`../examples/sample-morning-routine-prompt.md`](../examples/sample-morning-routine-prompt.md).

Privacy rule: do not publish your real Google Sheet IDs, API keys, email addresses, or local file paths.
