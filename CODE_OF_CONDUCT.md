# Code Of Conduct

This project is for beginners, non-developers, and builders learning in public. Be respectful, practical, and specific.

## Expected Behavior

- Use welcoming language.
- Give constructive feedback.
- Respect different skill levels.
- Assume good intent.
- Protect privacy and do not ask people to reveal private accounts, API keys, emails, or file paths.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Sharing someone else's private data.
- Gatekeeping beginners for not knowing technical terms.
- Spam or unrelated promotion.

## Enforcement

Maintainers may remove comments, close issues, or block users who violate this code of conduct. If you see a problem, open a private report through the maintainer's preferred contact channel.
