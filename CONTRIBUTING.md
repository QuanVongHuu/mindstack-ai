# Contributing

Cảm ơn bạn đã ghé! Mọi đóng góp, dù nhỏ, đều được trân trọng.

You do not need to be a developer to contribute. This repo is for people who want practical AI workflows, so wording fixes and real-world prompt adaptations are valuable.

## Ways To Contribute

- Share your adapted morning or evening prompt.
- Fix confusing wording.
- Translate a section into Vietnamese or improve an existing translation.
- Report a step that does not work for your setup.
- Suggest a new workflow, connector, or example.
- Add screenshots or diagrams.

## Easiest Way

1. Open the file you want to edit on GitHub.
2. Click the pencil icon.
3. Make your change.
4. Click "Propose changes".
5. Open a pull request.

Plain-language GitHub terms:

- Fork: your own copy of this repo.
- Branch: a temporary workspace for your change.
- Pull request: a request to merge your change into this repo.
- Issue: a place to report bugs, confusing steps, or ideas.

## Prompt Contributions

If you share a prompt adaptation, please include:

- What problem it solves.
- Which tools/connectors it uses.
- Which placeholders users must replace.
- Any privacy warning, especially for file paths, sheet IDs, email addresses, or API keys.

Never submit private Google Sheet IDs, API keys, real email addresses, or personal file paths unless they are fake examples.

## Labels

Beginner-friendly issues use:

- `good first issue`
- `help wanted`
- `translation`
- `prompt adaptation`

## Review Style

Be specific and kind. Prefer "cảm ơn bạn, mình gợi ý chỉnh chỗ này..." over "this is wrong".

Maintainer expectation: I will try to respond within about 10 days.
