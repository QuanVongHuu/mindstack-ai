# FAQ

## What Is mindstack-ai?

mindstack-ai is a beginner-friendly personal AI agent system. It starts with a no-code setup, then adds optional low-code tools when you need local file control or external app automation.

## Do I Need To Know Code?

No. Start with Claude Cowork and connectors. OpenCode + MCP/Composio is optional.

## Can I Use Notion Instead Of Obsidian?

Yes, but it is not a drop-in replacement. You can use Notion if your Claude, MCP, or Composio connector can read and update the right Notion pages or databases, and if you adapt the prompts to Notion's page/database structure. This repo defaults to Markdown/Obsidian because plain `.md` files are simple, portable, and local-first.

## Can I Remove Gmail Or Sheets?

Yes. Delete the sections you do not use and update the output template.

## Should I Publish My Real Prompt?

Only after removing private details. Replace real sheet IDs, local paths, names, and emails with placeholders.

## Why Use Exact Google Sheet IDs?

Exact IDs reduce ambiguity. Claude does not need to search Drive and accidentally pick the wrong file.

## Should The Main README Be English Or Vietnamese?

Use English for global discoverability and a full Vietnamese README for your local audience.
